## Getting Started

### Prerequisites
You must have [Node.js](https://nodejs.org/en/) downloaded.

This will enable you to install dependencies using npm. For example,

        npm install -g @angular/cli
        
### Understanding the Workflow

MongoDB : NoSQL database

Express : Back-end

Angular: Front-end

Node.js : Server
